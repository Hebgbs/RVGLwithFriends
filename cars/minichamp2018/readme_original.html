Build: New Car from scratch

Speed: 70 mph

Class: Super Pro

Description:

          Mini Champ is a GTP class car body, mounted on an RC carpet racer chassie. It's slightly smaller and faster than most Revolt cars. It carries three paint jobs, which can be switched with separate batch files. The first of three is my paint scheme, which can be used by running IBcar.BAT. The second is The Me & Me's paint scheme, which can be used by running M&Mcar.BAT. The third is a clear body paint scheme, that allows you to view the detailed chassie of the Mini Champ car. It can be used by running Clear.BAT.

Installation:

To install unzip into your Revolt folder.
