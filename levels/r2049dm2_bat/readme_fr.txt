Description
------------

«Melee» («Battle 2» dans la version N64 nord-américaine) est une des 8 arènes de bataille présentes dans les versions consoles de San Francisco Rush 2049.

Ce circuit inclut une version à l'envers «R» plus courte.

Inclut la musique originale des versions DC/MAT3 du jeux, 'Vice' par Barry Leitch.


Notes
------
	
	- Ce circuit n'est compatible qu'avec Re-Volt 1.2 ou RVGL et a été concu avec l'alpha 15.0325 du v1.2

	- Pour utiliser ce circuit en mode battle, simplement remplacer les fichiers originaux par ceux présents dans le dossier «battle_kit» mais n'oubiez pas de faire des sauvegardes des fichiers originaux d'abord !
